# Privacy Policy

**Last updated:** February 13, 2026

## Data Collection

This application accesses WHOOP health and fitness data **solely for the personal use of the account owner** (joe.giordano@a3inc.biz).

- No data is sold, shared, or used for advertising

## Data Storage

Authentication tokens are stored locally on the user's private systems. No WHOOP data is stored on external servers.
